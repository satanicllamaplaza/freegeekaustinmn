Secure Data Handling

Data security is very important to us. If people are kind enough to donate to us, we want to make sure that we protect thier information.

Offline

None of the donated materials will have internet access until ALL data has been securly destroyed. Our wiping station will only ever go online to recieve system updates once a week and never with an unprocessed drive connected.

Firmware Secure Erase Tools

Solid state drives will be wiped with their firmware secure erase tools. These are tools built into the SSD's firmware that trigger a complete wipe of all flash memory cells.

nwipe

We use Nwipe to securely erase sensitive data from storage devices by overwriting it with multiple passes of randomized information. This ensures that previously stored files cannot be recovered. The process follows recognized data sanitization standards.

Any storage devices that we can not securely wipe will be brought to a trusted recycling site where they will be physically destroyed.
